# dt和dd在同一行显示 #
1. 示例代码部分
```
`<dl>`
`<dt>test </dt>`
`<dd>t1 </dd>`
`</dl>`
`<dl>`
`<dt>test2 </dt>`
`<dd>t2 </dd>`
`</dl>`
```




2. style样式部分
```
`<style>`
`dl{`
`clear:left;`
`}`
`dt,dd{`
`float:left;`
`}`
`</style>`
```
由于markdown语法限制，复制时，请把`符号去除。
